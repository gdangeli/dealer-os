### Churn Calculation

```
Logo Churn Rate = Customers Lost / Starting Customers
Revenue Churn = MRR Lost / Starting MRR
Net Churn = (MRR Lost - Expansion MRR) / Starting MRR
NRR = (Starting MRR - Churned + Expansion) / Starting MRR
```
